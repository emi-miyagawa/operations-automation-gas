# Email Automation

Generates email drafts from spreadsheet data using Google Apps Script.

## Purpose
Reduce manual email creation and enable batch processing.

## Key Features
- Draft-based generation (allows manual validation)
- Structured input from Google Sheets
- Consistent formatting

## Impact
- Reduced processing time (~2–3 min → <30 sec per email)
- Enabled batch handling (30–50 emails/session)

## Trade-offs
- Not fully automated (manual review required)
- Prioritized flexibility over full automation
